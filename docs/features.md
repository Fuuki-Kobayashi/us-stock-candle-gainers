# US Stock Candle Gainers - 機能仕様書

## 1. 概要

US Stock Candle Gainers は、米国株式のローソク足パターン認識とリスク分析を行うWebアプリケーションです。FastAPI バックエンドと静的フロントエンドで構成され、以下の主要機能を提供します。

- 60種類のローソク足パターン自動検出（確定・予測）
- 財務リスク分析（財務健全性・オファリングリスク）
- 複数銘柄の一括スキャン（スクリーナー）
- パターン逆引き検索
- AIトレードプロンプト生成

---

## 2. 機能一覧

| # | 機能名 | エンドポイント | 説明 |
|---|--------|---------------|------|
| 1 | ローソク足パターン分析 | `POST /analyze` | 個別銘柄のローソク足パターンを検出・表示 |
| 2 | 財務リスク分析 | `POST /risk` | 財務健全性とオファリングリスクを8指標で評価 |
| 3 | スクリーナー | `POST /screener` | 最大50銘柄の一括パターンスキャン |
| 4 | パターン検索（逆引き） | `GET /api/patterns`, `POST /api/pattern-search` | パターンIDを指定して該当銘柄を検索 |

| 6 | フロントエンドUI | `GET /`, `GET /screener`, `GET /pattern-search` | 3ページ構成のダークテーマUI |

## 3. 各機能の詳細説明

### 3.1 ローソク足パターン分析

#### ユーザーストーリー

> 投資家として、指定した銘柄の直近ローソク足からパターンを自動検出し、買いシグナル・売りシグナルを把握したい。また、シミュレーションモードで仮想的な値動きに対するパターンも確認したい。

#### 4つの分析モード

| モード | 条件 | 説明 |
|--------|------|------|
| `realdata` | change1〜change3 未指定、candle_count=3（デフォルト） | 実際のOHLCVデータ3本を取得し、3本足パターンを検出 |
| `realdata_2candle` | change1〜change3 未指定、candle_count=2 | 実際のOHLCVデータ2本を取得し、2本足パターンを検出 |
| `simulation_predicted` | change1 + change2 を指定 | 直近終値を基準に2本のシミュレーションローソク足を生成し、3本目を予測 |
| `simulation_confirmed` | change1 + change2 + change3 を全て指定 | 3本のシミュレーションローソク足を生成し、確定パターンを検出 |

#### 動作フロー

1. ユーザーがティッカーシンボルを入力（例: `AAPL`）
2. モードに応じてデータを取得またはシミュレーション生成
3. パターン検出エンジンが60パターンに対してマッチングを実行
4. 検出結果（確定パターン/予測パターン）をレスポンスとして返却

#### 画面表示要素

- **Canvasローソク足チャート**: OHLCVデータを視覚的に描画
- **OHLCVテーブル**: 日付、始値、高値、安値、終値、出来高を表形式で表示
- **パターン一覧**: 検出されたパターンの名前、シグナル、説明、方向を表示
- **ショートインタレスト情報**: 空売り比率、ショートレシオ、空売り株数等（月次更新）
- **ATR(14)**: Average True Range（14日間）の値を表示

#### パターンの種類

- **確定パターン（confirmed）**: 必要な全てのローソク足が揃い、パターンが確定した状態
- **予測パターン（predicted）**: 2本目までのローソク足から3本目の条件を予測した状態

### 3.2 財務リスク分析

#### ユーザーストーリー

> 投資家として、特定の銘柄の財務健全性とオファリングリスクを定量的に把握し、投資判断の参考にしたい。

#### 評価指標

##### 財務健全性（Financial Health）

| 指標 | 説明 |
|------|------|
| D/Eレシオ（負債資本比率） | 負債と自己資本の比率。低いほど健全 |
| 流動比率（Current Ratio） | 短期支払能力の指標。1.0以上が望ましい |
| 利益率（Profit Margin） | 売上に対する純利益の割合 |
| フリーキャッシュフロー（FCF） | 事業活動から得られる自由な現金の量 |

##### オファリングリスク（Offering Risk）

| 指標 | 説明 |
|------|------|
| キャッシュランウェイ（Cash Runway） | 現在の現金燃焼率での事業継続可能期間 |
| 希薄化リスク（Dilution Risk） | 新株発行による既存株主の持分希薄化リスク |
| 純利益（Net Income） | 最終的な利益額 |
| 負債現金比率（Debt/Cash Ratio） | 負債に対する手元現金の比率 |

#### リスクレベル

各指標は以下の3段階で評価されます。

| レベル | 意味 |
|--------|------|
| 低 | リスクが低い（良好） |
| 中 | 注意が必要 |
| 高 | リスクが高い（要注意） |

#### 動作フロー

1. ユーザーがティッカーシンボルを入力
2. yfinance APIから財務データを取得
3. 8つの指標それぞれについてリスクレベルを算定
4. 財務健全性・オファリングリスクそれぞれの総合評価とサマリーを生成

### 3.3 スクリーナー

#### ユーザーストーリー

> 投資家として、複数の銘柄を一括でスキャンし、ローソク足パターンが検出された銘柄を効率的に発見したい。

#### 仕様

- **最大銘柄数**: 50ティッカー
- **ローソク足本数**: 2本モードまたは3本モード（デフォルト: 3本）
- **出力**: 各銘柄ごとのパターン検出結果、変動率、エラー情報

#### 動作フロー

1. ユーザーが複数のティッカーシンボルを入力（カンマ区切り等）
2. 各銘柄に対してパターン検出を実行
3. 結果を一覧表示（パターンが検出された銘柄をハイライト）
4. エラーが発生した銘柄はエラー情報とともに表示

#### レスポンス集計

| フィールド | 説明 |
|-----------|------|
| total | リクエストされた銘柄の総数 |
| scanned | 正常にスキャンが完了した銘柄数 |
| errors | エラーが発生した銘柄数 |

---

### 3.4 パターン検索（逆引き）

#### ユーザーストーリー

> 投資家として、特定のローソク足パターン（例: 明けの明星）を探しており、どの銘柄でそのパターンが出現しているかを知りたい。

#### 2つのエンドポイント

##### パターン一覧取得（GET /api/patterns）

全60パターンのレジストリを取得します。各パターンには以下の情報が含まれます。

- パターンID（機械可読な識別子）
- パターン名（日本語）
- 方向（bullish / bearish）
- 利用可能な検出タイプ（confirmed / predicted）
- ローソク足本数（1 / 2 / 3）

##### パターン逆引き検索（POST /api/pattern-search）

指定したパターンIDに一致する銘柄を、最大50銘柄からスキャンして検索します。

#### 動作フロー

1. ユーザーがパターン一覧からパターンを選択
2. 対象銘柄リストを入力
3. 各銘柄に対してパターンマッチングを実行
4. 一致した銘柄とパターン詳細を表示

---

### 3.5 AIトレードプロンプト生成

#### ユーザーストーリー

> 投資家として、特定銘柄の総合的なトレード分析プロンプトを生成し、外部のAI（ChatGPT等）にコピー＆ペーストしてトレード戦略の提案を受けたい。

#### 仕様

- **非同期並列データ取得**: 企業プロフィール、テクニカルデータ、リスク分析、ショートインタレスト、ニュースを並列で取得
- **出力形式**: Markdown形式のトレード分析プロンプト
- **外部AI API呼び出しなし**: プロンプトテキストの生成のみを行い、AI APIの呼び出しは行わない
- **コピー機能**: モーダルUI内でクリップボードにコピー可能

#### 動作フロー

1. ユーザーが銘柄のAIトレードプロンプトボタンをクリック
2. バックエンドが非同期で各種データを並列取得
3. Markdownプロンプトを生成してレスポンスとして返却
4. モーダルUIにプロンプトを表示
5. ユーザーが「コピー」ボタンでクリップボードにコピー
6. 外部AIサービスにペーストして分析を依頼

---

### 3.6 フロントエンドUI機能

#### ページ構成

| ページ | パス | ファイル | 説明 |
|--------|------|---------|------|
| メインページ | `/` | `static/index.html` | 個別銘柄分析・リスク分析・AIプロンプト |
| スクリーナー | `/screener` | `static/screener.html` | 複数銘柄一括スキャン |
| パターン検索 | `/pattern-search` | `static/pattern-search.html` | パターン逆引き検索 |

#### UI特徴

- **ダークテーマ**: ノイズテクスチャ + スキャンラインエフェクト
- **Canvas描画ローソク足チャート**: HTML5 Canvasによるローソク足の描画
- **パターン詳細モーダル**: Canvas描画付きのパターン詳細表示
- **パターンフィルタリング**: 全て / 確定 / 予測 の3つのフィルター
- **画像保存機能**: html2canvasを使用した「画像として保存」機能
- **URLパラメータ対応**: `?ticker=AAPL` のようなURLパラメータでの銘柄指定
- **ページ間ナビゲーション**: 各ページ間のリンク
- **レスポンシブデザイン**: Tailwind CSSによるレスポンシブ対応
